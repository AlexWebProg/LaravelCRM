## LaravelCRM
CRM на Laravel 9.47.0 + jQuery 3.6.0 + AdminLTE 3.2
<br/>
Личный кабинет клиента и система управления клиентами

### Инструменты:
- Policy
- Middleware
- Job
- Service
- Mail notification
- Laravel Excel
- Custom validation rule

### Интеграции:
- TinyPNG
- PSPDFKit
- ConvertAPI
- iLovePDF
- MediaIO
- GoogleSheets
